AI-Powered Snowflake Blog AI Assistant

Use Case: Chat-Based Blog Search

Overview:

A LLM Chatbot that uses Retrieval Augmented Generation to answer questions about a particular blog or a list of blogs from snowflake technology.

Scenario:

A user can search the blogs from snowflake and retrieve the answers related with data technology

Technologies Used:

Powered by OpenAI GPT-4-1106-preview LLM, LlamaIndex, with prompt engineering techniques, deployed on Streamlit Cloud, and developed using Python. Access requires an OpenAI API Key.

Note: You will need an OpenAI API key to access the chatbot, which you can obtain from OpenAI API Keys.
