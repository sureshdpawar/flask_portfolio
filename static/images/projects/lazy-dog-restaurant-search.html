Welcome to the Search Assistant of Lazy Dog Restaurant

Developed using the Generative AI Search conversational App feature of Google Cloud's Vertex AI.

You can search the restaurant website https://www.lazydogrestaurants.com/ for locations, menu items .

Use Case: Conversational Search for Restaurant Information

Overview:

Implementing Google's Vertex AI Conversational Search, this application transforms how users interact with a restaurant website. It allows for intuitive, conversational queries to explore restaurant locations, menus, and other features.

Scenario:

A user looking for a specific cuisine or dish uses the app to ask, "what are locations?" or "show me burgers." The app, powered by Vertex AI, comprehensively understands these queries and fetches relevant information.

Technologies Used:

  • Google Vertex AI Conversational Search: For natural language query processing.
  • Python: Backend development and data interfacing.
  • Flask (Python Web Framework): Server-side request handling.
  • HTML/CSS/JavaScript: User interface development.
  • Restaurant website: Real-time information access.

Outcome:

The application enhances the user experience on the restaurant website, enabling users to find information through conversational questions, increasing customer satisfaction and retention.
