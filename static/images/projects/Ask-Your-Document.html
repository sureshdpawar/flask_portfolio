Ask-your-documents with LLM, Langchain, VectorDB

"Ask-Your-Documents" is an innovative project designed to revolutionize how users interact with extensive document databases. Leveraging the power of Large Language Models (LLM), Langchain, and VectorDB, this project provides an intuitive query interface where users can ask natural language questions and receive accurate, context-aware responses. By integrating these technologies, the system analyzes and understands vast amounts of text data, offering precise answers extracted from a comprehensive document repository.

Use Case: Search documents for real time answers

Objectives:

To provide a seamless interface for querying large document sets using natural language.

Technologies Used:

  • ChromaDB Vector Database: Efficient retrieval of relevant content.
  • Large Language Model (LLM) databricks/dolly-v2-3b: Powers real-time content generation.
  • Langchain : Langchain for in-prompt queries
